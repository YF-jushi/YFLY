# 参数

- 主控：STM32F405RGT6
- IMU：MPU9250（加速度计/陀螺仪/磁力计）
- 气压计：SPL06
- GPS：M8N（内置HMC5883L磁力计）
- 黑匣子：TF卡
- 光流接口：支持正点原子光流模块
- 扩展接口：SPI x 1 ; UART x 2
- 解码：PPM/SBUS编码
- 数传：单纯的串口（本人使用的是无线仿真器，有仿真和串口功能）
- 支持地面站：匿名地面站/QGC
